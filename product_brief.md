# Task 1: Product Scoping
## Marketing Performance Dashboard

### Problem Statement
The team manually digs through multiple tools 
to answer: "How is our marketing performing 
across channels right now, and where should 
we be focusing?"

This takes too long, looks different every time, 
and depends on one person.

### Solution
Build an internal dashboard that automatically 
consolidates marketing KPIs from all channels 
in one place.

### Primary User
Internal marketing analysts at Tacheon

### What the Tool Does (V1)
- Shows KPIs: impressions, clicks, conversions, spend
- Compares performance across channels
- Highlights best and worst performing channel
- Date range filter (last 7, 14, 30 days)

### Data Sources
- Google Analytics (website traffic)
- Meta Ads (Facebook/Instagram)
- Google Ads (paid search)

### What is NOT in V1
- Real-time updates
- Client-facing access
- AI recommendations
- Custom report builder

### Why These Decisions
Keeping V1 simple ensures the tool gets used. 
Adding too many features risks building something 
nobody trusts or understands.
